## UK Train Stations Power BI Project
This project uses ticket transaction data from train stations in the United Kingdom to create interactive visualizations with Power BI. 

## Project Overview
The dataset consists of ticket transaction data, which has been modeled and visualized to provide insights into:
- Ticket sales performance per station
- Distribution of ticket types
- Temporal trends in ticket purchases

## Tools & Technologies
- Microsoft Power BI Desktop
- Data Modeling (Star Schema)
- Interactive Visualizations & DAX

## Installation Instructions
- Download and open the file using Microsoft Power BI Desktop.
- Explore the dashboards and visualizations that have been created.

## Usage Instructions
- Use the available slicers to filter the data based on date, station, or ticket type.
- Navigate the map to view sales per station and see additional information through tooltips.

## Data Source
- The data source is: https://mavenanalytics.io/data-playground, while the development of the Data Model was created by Data Tutor (https://datatutor.gr).
- Finally, this report (specifically the Data Model part) was created by following the Guided Project course in Microsoft Power BI by Data Tutor.

## Key Insights
- The data shows a total revenue of 741K from 31,619 tickets over the period December 2023 - April 2024 with an average price of £23.44. This indicates a relatively affordable pricing structure.
- Advance-ticket type has the highest share (42%), suggesting that a significant portion of customers prefer booking in advance to secure lower prices.
- Revenues appear relatively stable after mid-March, due to the normalization of travel patterns.
  
## Business Recomendations
- Develop targeted strategies to improve under-performing stations.
- Promote off-peak tickets to balance passenger distribution and reduce congestion.
