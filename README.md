## UK Train Stations Power BI Project
This project uses ticket transaction data from train stations in the United Kingdom to create interactive visualizations with Power BI. The .pbix file contains various charts, map, and slicers for analyzing ticket sales by station and by ticket type. 
The data source is: https://mavenanalytics.io/data-playground, while the development of the Data Model was created by Data Tutor (https://datatutor.gr).
Finally, this report (specifically the Data Model part) was created by following the Guided Project course in Microsoft Power BI by Data Tutor.

## Installation Instructions
Open the file using Microsoft Power BI Desktop.
Explore the dashboards and visualizations that have been created.

## Usage Instructions
Use the available slicers to filter the data based on date, station, or ticket type.
Navigate the map to view sales per station and see additional information through tooltips.
